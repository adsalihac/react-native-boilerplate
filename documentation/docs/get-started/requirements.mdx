# Requirements {#requirements}

This react native boilerplate is built with the following technologies:

- This project is compact with macOS , So you need Apple Macbook or iMac to run this project. Otherwise you need to manual changes in the project.

- React Native CLI

  <span>{constants.reactNativeVersion}</span>

- BoilerPlate Version

  <span>{constants.boilerplateVersion}</span>

- The minimum OS versions the app will run on are
  [Android 5.0 (SDK 21)](https://developer.android.com/studio/releases/platforms#5.0)
  and
  [iOS 12.4](https://developer.apple.com/documentation/ios-ipados-release-notes/ios-12_4-release-notes).
- The maximum OS versions the app will run on are
  [Android 13 (SDK 33)](https://developer.android.com/about/versions/13) and
  [iOS 16](https://developer.apple.com/ios/).

# Getting Started

- Set up your development environment by following the
  [React Native docs](https://reactnative.dev/docs/environment-setup).

- Install the correct version of CocoaPods by installing
  [podenv](https://github.com/kylef-archive/podenv) and running
  `npm run install:pod`.

- Install the correct version of node.js by installing
  [nvm](https://github.com/creationix/nvm) and running `nvm install` in the
  project root.

- Install the node modules by running `npm install` and wait for everything to
  be built successfully.

:::tip
This documentation is prepared orderly. You can start from the first step and go , please do not skip any steps. If you have any questions, please contact us.
:::
