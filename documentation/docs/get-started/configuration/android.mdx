in android. it can easy to multiple apk in different [environment](#configuration-environment-files).

## Steps

1. First goto `android/app/build.gradle`

```bash
flavorDimensions 'branding'
productFlavors {
    brand1 {
        dimension 'branding'
    }
}
```

- You can replace above code.

2 - Goto `app/navigators` , you can see the `brand1` folder , change the folder name to your brand name <AppBrand background={true}/>.

3 - Goto `app/screens` , you can see the `brand1` folder , change the folder name to your brand name <AppBrand background={true}/>.

4 - Open `app/app.tsx` , replace below lines with your brand name <AppBrand background={true}/> .
