# Flipper

## Installation

- Go to [Flipper Releases](https://github.com/facebook/flipper/releases) and
  install the latest version.

:::note
Change the Flipper configuration in `android/gradle.properties` and
`ios/Podfile` with the version number
:::

Current version is used

<p>{constants.flipperVersion}</p>

## Execution

With flipper running, just run the app and enable or install some extensions.

For redux debugging to work, you will need to install
[flipper-plugin-redux-debugger](https://github.com/jk-gan/flipper-plugin-redux-debugger).
